# HireSage Brand Kit

## Brand Overview

**HireSage** is an AI-powered career assistant that helps job seekers analyze roles, generate applications, prepare for interviews, and manage their job search pipeline.

The brand represents:

* wisdom
* clarity
* intelligent guidance
* professional confidence

The tone should feel like **a trusted advisor rather than a flashy AI tool**.

---

# Brand Positioning

### Mission

Help professionals land better jobs faster using intelligent tools.

### Product Category

AI-powered job application assistant.

### Value Proposition

HireSage transforms the job search into a structured, intelligent process.

### Tagline Options

Primary:

**Apply with wisdom.**

Alternatives:

* Your AI job search advisor
* Land your next role smarter
* Intelligent job search
* Your AI co-pilot for job applications

---

# Brand Personality

| Trait        | Description                          |
| ------------ | ------------------------------------ |
| Intelligent  | thoughtful AI assistance             |
| Calm         | confident, not hype-driven           |
| Professional | career-focused                       |
| Efficient    | reduces friction in job applications |

Avoid:

* gimmicky AI branding
* playful cartoon design
* gaming aesthetics

The product should feel closer to **Notion, Linear, or Stripe** than consumer apps.

---

# Logo System

## Primary Logo

Structure:

```
[Icon] HireSage
```

The icon should be simple and scalable.

### Recommended icon concepts

1. Stylized **S arrow**
2. Minimal **compass / guidance mark**
3. Abstract **neural spark**
4. Minimal **owl-inspired geometry**

Avoid literal animals or overly detailed shapes.

---

## Logo Rules

Clear space:

```
1x height of the “H”
```

Minimum size:

```
120px width
```

For favicons and extensions, use **icon-only version**.

# Implementation Note for Engineers

Do not hardcode colors.

Create semantic tokens such as:

```
--color-primary
--color-accent
--color-text
--color-border
```

This ensures design consistency across the application.
